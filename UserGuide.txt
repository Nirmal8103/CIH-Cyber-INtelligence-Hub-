Important Commands
To initiatiate venv: 
cd C:\Users\Your_Name\folder\folder\your_project_name
python -m venv env

To start venv:
env\Scripts\activate
To install requirements:
pip install -r requirements.txt

To start project:
python -m src.visualizer.dashboard

To scrape news:
python -m src.scraper.scraper
or
python src/scraper/scraper.py
